# BANILA CO — Brand Identity Guidelines (Agent Reference)
## ⚠️ SOURCE: 공식 Product Visual Guide (Directional Moodboard & Visual Standard) 기반

## Brand Essence
**"Zero to Glow"** — 모든 아름다움은 제로(클렌징)에서 시작되어 글로우(빛나는 피부)로 완성된다.

## Brand Promise
"The First Step to Your Best Skin"

## VI Keywords (공식)
**핑크 / 여유있고 정돈된 자신감 / Refined / Urban Feminine**

## Brand Archetype
- **Primary: The Caregiver** — 피부를 진심으로 돌보고, 모든 피부 타입을 존중
- **Secondary: The Sage** — 18년간 6세대 R&D, Zero Balance Technology, 과학적 전문성
- **Shadow: The Creator** — 위트와 감성, 의미 있는 놀이(Meaningful Play)

## Proof Points
- Heritage: K-Beauty 클렌징 밤의 원조, 2005년부터
- Science: 6세대 Zero Balance Technology
- Scale: 글로벌 1억 개 판매, 22% 시장 점유율
- Clean: 파라벤 프리, EU 비건 인증, 식물 유래 성분

## Taglines (Market-Specific)
- Global EN: "From Zero to Glow"
- Korea: "제로에서 시작되는 빛나는 피부"
- Japan: "ゼロから始まる、あなたの輝き"
- China: "从零开始，绽放光彩"

## TARGET USERS (공식 Product Visual Guide)

### ① Core User (실제 구매·소비 중심층)
- **연령**: 24–26세
- **특성**: 트렌드 감도 높음, 디자인 완성도와 브랜드 태도 중시

### ② Aspiration User (브랜드가 지향하는 "언니" 포지션)
- **연령**: 28–29세
- **역할**: Core User가 동경하는 "한 단계 위의 라이프스타일"
- **이미지**: 도시적·정제된 언니 이미지, 세련된 자기 관리의 상징
- **⚠️ 모든 비주얼·패키지·공간의 기본값 = ②번 Aspiration User**

### ③ Model / Visual Representation
- **연령**: 17–23세
- **역할**: 시각적 신선함과 생기 제공, 스타일링·무드 연출로 성숙도를 조절
- 우리의 룩앤필, 추구미

### 경쟁사 포지셔닝
- **vs fwee, Amuse**: 더 정제된 톤, 더 높은 밀도감의 디자인
- "귀여움"이 아닌 "세련됨"에 방점

---

## WHAT WE DON'T DO (공식 배제 기준)

절대 하지 않는 것:
- ❌ **"뀨"**, 키치, 과도한 러블리 연출
- ❌ 과도한 장식 요소
- ❌ **10대 감성으로 오해될 수 있는 톤앤매너**
- ❌ 즉각적인 **유행어·밈 중심 커뮤니케이션**
- ❌ 과도한 네온 컬러
- ❌ VI 키워드 외의 톤: cute / girly / kitsch / loud / trendy slang

감사 시 이 배제 기준 위반은 **Critical** 등급으로 처리합니다.

## COLOR SYSTEM (공식 Product Visual Guide P8 기반)

### 3-Tier Color System

#### 메인 컬러 (기준 컬러)
브랜드의 일관된 이미지를 위해 우선 적용되는 기준 컬러.
더 정제된 톤과 높은 밀도감의 디자인으로, "귀여움"이 아닌 "세련됨"을 중심으로 한 바닐라코의 기준 컬러.

| Role | Description | Usage |
|------|------------|-------|
| Main Pink | 바닐라코 시그니처 핑크 | 로고, 대표 비주얼, 패키지 기본 |

#### 밝은 확장 컬러 (Light Extension)
브랜드 메인 컬러를 환하게 확장한 컬러 계열로, 공간을 맑고 가볍게 연출.

| Role | Description | Usage |
|------|------------|-------|
| Light Pink | 밝고 가벼운 인상의 확장 | 배경, 공간, 디지털 서피스 |

#### 깊은 확장 컬러 (Deep Extension)
성숙하고 깊이있는 확장 컬러. "언니미"가 느껴지는 오피니언 리더 무드를 표현.
고급스러움이 요구되는 마케팅 커뮤니케이션 영역에서 사용.

| Role | Description | Usage |
|------|------------|-------|
| Deep Pink/Berry | 고급스러운 딥 확장 | 마케팅, 프리미엄 커뮤니케이션 |

#### 시선을 끄는 강조 컬러 (Accent)
온라인, 포스터, 패키지 일부 강조 영역에 사용.
과도한 네온 컬러가 아닌, 메인 컬러보다 채도와 임팩트를 한 단계 높인 컬러.

| Role | Description | Usage |
|------|------------|-------|
| Accent | 채도 높은 강조 컬러 | 온라인 배너, 포스터, 패키지 강조 |

### Product Line별 컬러 시스템
각 제품 라인은 독립적 무드보드와 컬러를 가지며, 감사 시 해당 라인의 기준을 적용해야 합니다:

| 라인 | 컬러 톤 | 무드 키워드 | Pantone |
|------|---------|-----------|---------|
| **Core (클린잇제로)** | 시그니처 핑크 | "Pink Denim" — 밝고 캐주얼, 일상 감성 | — |
| **Core (클린/그린)** | 세이지 그린 | 클린 뷰티, 자연+과학 | — |
| **Airy Blossom** | 핑크+라벤더 그라디언트 | 몽환적, 에어리, 소프트 스튜디오 | — |
| **Prime Primer** | 실버/모노톤 | B&W, 시크, 권위적, 프로페셔널 | 000 C |
| **Sheer Blur (컬러)** | 레드/코랄 | 대담한 색조, 컬러풀, 임팩트 | 94 C |

### 컬러 사용 규칙 (공식)
- 메인 컬러가 항상 우선 적용 — 브랜드 일관성의 기준
- 밝은 확장 컬러는 공간·배경에 맑고 가볍게
- 깊은 확장 컬러는 프리미엄·언니미 영역에 한정
- 강조 컬러는 과도하지 않게 — 네온 금지, 메인 대비 한 단계 높인 채도
- **컬러가 실제 작업 환경과 다르게 보일 수 있으므로 컬러 넘버를 반드시 확인**
- 라인별 컬러를 혼합 사용 금지 (예: Prime Primer의 실버 톤을 Airy Blossom에 적용 ❌)

## IMAGERY & PHOTOGRAPHY (공식 Visual Guide 기반)

### Visual 기본값
**모든 비주얼의 기본값 = ② Aspiration User (28-29세)**
- 도시적, 정제된, 세련된 자기 관리의 이미지
- "귀여움"이 아닌 "세련됨" 중심

### Product Line별 포토 무드
| Line | 무드 | 촬영 스타일 | 키워드 |
|------|------|------------|--------|
| Core (Pink Denim) | 밝고 캐주얼, 일상 감성 | 핑크 톤, 데님 질감, 라이프스타일 | 일상, 신선, 트렌디 |
| Core (Green/Clean) | 클린 뷰티, 자연+과학 | 세이지 그린, 미니멀, 클린 | 자연, 과학, 깨끗 |
| Airy Blossom | 몽환적, 에어리 | 핑크+라벤더 그라디언트, 소프트 스튜디오 | 몽환, 가벼움, 블러 |
| Prime Primer | 시크, 권위적 | 실버/B&W, 모노톤, 하이컨트라스트 | 프로, 시크, 도시 |
| Sheer Blur (Color) | 대담한 색조 | 레드/코랄, 컬러풀, 클로즈업 | 컬러, 임팩트, 표현 |

### Photo Style (전체 공통)
- **Lighting**: Natural, soft, golden-hour preferred. No harsh flash.
- **Color Temp**: Warm (slightly warm white balance)
- **Composition**: Clean negative space, rule of thirds
- **Focus**: Shallow depth of field for product shots, sharp for lifestyle
- **Retouching**: Minimal. Preserve natural skin texture. No over-smoothing.
- **여성적인 라운드 무드**: 에지/무드를 부분적으로 조합해 톤을 다채롭게
- **고급한 인상 유지**: B컷 미연출로 제품 중심의 시선이 가는 공간을 형성

### Model Casting (공식 기준)
- **모델 연령대**: 17-23세 (시각적 신선함 + 생기)
- 스타일링·무드 연출로 성숙도를 조절
- 자연스럽고 건강한 피부 (과보정 금지)
- 글로벌 시장: diverse skin tones, ethnicities 고려

### Image Don'ts
- ❌ 과도한 보정/에어브러시
- ❌ 피부톤 밝게 보정 (화이트워싱)
- ❌ 비현실적 before/after 과장
- ❌ 경쟁사 제품 비교 이미지
- ❌ 저화질/픽셀화된 이미지
- ❌ **10대 감성으로 오해될 수 있는 연출** (키치, 뀨, 과도한 러블리)
- ❌ 과도한 장식 요소
- ❌ 라인별 무드 혼합 (예: Prime Primer의 B&W를 Airy Blossom에 적용)

## LAYOUT & SPACING SYSTEM (P1)

### 12-Column Grid
- **Desktop** (≥1200px): 12col, gutter 24px, margin 64px
- **Tablet** (768–1199px): 8col, gutter 20px, margin 32px
- **Mobile** (≤767px): 4col, gutter 16px, margin 16px
- 콘텐츠 최대 너비: 1200px
- 그리드 밖 요소 배치 금지

### 8px Spacing Scale
| Token | Value | Usage |
|-------|-------|-------|
| sp-1  | 4px   | 인라인 간격, 아이콘-텍스트 |
| sp-2  | 8px   | 요소 내부 패딩 최소 |
| sp-3  | 12px  | 관련 요소 간 간격 |
| sp-4  | 16px  | 카드 내부 패딩, 리스트 간격 |
| sp-5  | 24px  | 섹션 내 그룹 간격 |
| sp-6  | 32px  | 카드 내부 대형 패딩 |
| sp-7  | 48px  | 섹션 간 간격 |
| sp-8  | 64px  | 대형 섹션 분리 |
| sp-9  | 96px  | 페이지 섹션 간 마진 |
| sp-10 | 128px | 히어로/풋터 패딩 |

### Spacing Rules
- 모든 간격은 8의 배수 (4px는 미세 조정용 예외)
- 임의 값(13px, 37px 등) 사용 금지
- 패딩과 마진에 일관된 토큰 사용

## UI/UX PATTERNS (P3)

### Layout Patterns
- **Z-Pattern**: 랜딩 페이지, 제품 상세 (좌상→우상→좌하→우하 시선 흐름)
- **F-Pattern**: 블로그, 성분 설명 페이지
- **Card Grid**: 제품 목록, 컬렉션 (2-4열 반응형)
- **Hero + Content**: 캠페인 페이지 (풀스크린 히어로 → 스크롤)

### Navigation Rules
- 글로벌 네비: 최대 7개 메뉴 항목
- 모바일: 햄버거 메뉴 + 하단 탭바(4-5항목)
- 브레드크럼: 3뎁스 이상 페이지에 필수
- 검색: 항상 접근 가능, 우측 상단 배치

### Empty / Error / Loading States
| State   | Design Rules |
|---------|-------------|
| Empty   | 브랜드 일러스트 + 행동 유도 CTA. "아직 장바구니가 비어있어요" 스타일의 따뜻한 문구 |
| Error   | Error 컬러(#D4545C) 아이콘 + 구체적 원인 설명 + 해결 방법. "다시 시도" CTA |
| Loading | Rose 계열 스켈레톤 UI (Blush Light → White 펄스 애니메이션). 3초 이상 시 프로그레스 표시 |
| 404     | 브랜드 무드의 일러스트 + "길을 잃었나요?" + 홈/검색 CTA |

### Micro-Interactions
- **Page Transition**: fade + translateY(20px), 0.3s ease
- **Button Hover**: scale(1.02), 0.2s
- **Card Hover**: translateY(-4px) + shadow-lg, 0.4s
- **Color Swatch Click**: 복사 피드백 토스트, 1.2s auto-dismiss
- **Scroll Reveal**: opacity 0→1, translateY(30px→0), threshold 10%
- **No jarring animations**: 갑작스러운 확대, 번쩍임, 과도한 바운스 금지

- [ ] 위시리스트 / 즐겨찾기
- [ ] 키보드 단축키 (데스크톱)
- [ ] 제품 비교 기능

### 8. Aesthetic and Minimalist Design (미학적 미니멀 디자인)
- [ ] 불필요한 장식 요소 없음
- [ ] 정보 위계가 명확 (시각적 우선순위)
- [ ] 여백(white space) 충분
- [ ] 한 화면에 하나의 핵심 메시지

### 9. Help Users Recognize, Diagnose, and Recover from Errors (오류 복구)
- [ ] 에러 메시지가 구체적 ("이메일 형식이 올바르지 않습니다" vs "오류")
- [ ] 해결 방법 제시
- [ ] 에러 상태에서도 입력 데이터 보존

- [ ] FAQ / 도움말 접근 용이
- [ ] 제품 사용법 안내
- [ ] 챗봇/CS 연결
- [ ] 검색 기능 효과적

### Cognitive Load Assessment (인지 부하 평가)
- **시각적 복잡도**: 한 화면에 시각 요소 7±2개 이내

감사 결과 심각 이슈 발견 시, 두 가지 대안 방향을 제시:
- **Direction A**: 현 디자인 기반 최소 수정 (Quick Fix)

### Severity Levels
- **Critical** 🔴: 브랜드 오인 위험. 즉시 수정 필요
- **Major** 🟠: 가이드라인 명확한 위반. 다음 업데이트 시 수정
- **Minor** 🟡: 가이드라인 취지와 약간 벗어남. 개선 권장
- **Suggestion** 🟢: 위반 아님, 더 좋아질 수 있는 제안
